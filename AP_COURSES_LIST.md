## Usage in Application

When students select AP courses, they should choose from this complete list with:
- Subject name (from above)
- Score (1-5)
- Year taken (9th, 10th, 11th, 12th)

This allows the system to:
1. Check major alignment (e.g., CS major should have AP CS A, Calc BC)
2. Assess curriculum rigor
3. Evaluate course progression
4. Calculate average AP score
